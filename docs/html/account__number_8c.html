char * generate_unique_account_number ( void )

Generates a unique 8-digit account number.

This function repeatedly generates a random 8-digit numeric string until it finds one that is not "00000000" and does not already exist in the CSV file. The result is stored in a static buffer and returned as a string.

Returns
A pointer to a unique 8-digit null-terminated account number string.
